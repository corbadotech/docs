---
title: "Write Authorization header"
description: "Learn how to write the Authorization header to pass the session-token"
---

Writing the `Authorization` header is usually quite easy:

<CodeGroup>
```javascript JavaScript fetch API
const sessionToken = 'xxx';

fetch('https://api.example.com', {
  headers: {Authorization: `Bearer ${sessionToken}`}
})
.then(console.log).catch(console.log);
```

```javascript JavaScript Axios
const sessionToken = 'xxx';

Axios.post('https://api.example.com', {}, {
  headers: {Authorization: `Bearer ${sessionToken}`}
})
.then(console.log).catch(console.log);
```
</CodeGroup>
