## Set up CorbadoProvider

Integrate the [CorbadoProvider](/corbado-complete/frontend-integration/react) into the root of your application:

```tsx App.tsx
import { CorbadoProvider } from '@corbado/react';

function App() {
  return (
    <CorbadoProvider projectId="pro-1234567890">
      {/* Your application */}
    </CorbadoProvider>
  );
}

export default App;
```
