## Setup Corbado

As a prerequisite to all future actions, Corbado has to be loaded.
We do this by using the [Corbado.load](/ui-components/configuration/corbadoprovider) function.
We also maintain a global state, indicating whether Corbado is successfully loaded.

```html App.vue
<script setup lang="ts">
import Corbado from "@corbado/web-js"
import { ref, provide, onMounted } from 'vue'

// Create a global state
const isCorbadoLoaded = ref(false)

onMounted(async () => {
    await Corbado.load({
        projectId: "pro-XXX"
    })
    isCorbadoLoaded.value = true // Set the global state to true after Corbado is loaded
})

// Provide the state globally
provide('isCorbadoLoaded', isCorbadoLoaded)
</script>

<template>
    <!-- Your app content -->
</template>

```

<Note>
    Most customization is done in the `Corbado.load` function.
    See a full list of options for the `Corbado.load` function in the [web-js
    documentation](/ui-components/configuration/corbadoprovider).
</Note>
