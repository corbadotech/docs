## Set up CorbadoProvider

Integrate the [CorbadoProvider](/frontend-integration/react) into the root of your application:

```tsx App.tsx
import { CorbadoProvider } from '@corbado/react';

function App() {
  return (
    <CorbadoProvider projectId="pro-XXX">
      {/* Your application */}
    </CorbadoProvider>
  );
}

export default App;
```
