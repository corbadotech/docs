## Setup Corbado

Integrate the [CorbadoProvider](/ui-components/corbadoprovider) into the root of your application:

```tsx App.tsx
import { CorbadoProvider } from '@corbado/react';

function App() {
  return (
    <CorbadoProvider projectId="pro-XXX">
      {/* Your application */}
    </CorbadoProvider>
  );
}

export default App;
```

<Note>
    Most customization is done in the `CorbadoProvider`.
    See a full list of options for the `CorbadoProvider` in the [CorbadoProvider documentation](/ui-components/corbadoprovider).
</Note>
