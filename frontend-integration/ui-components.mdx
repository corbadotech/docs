Corbado helps you to add a full passkey authentication process to your application. As all of this involves a lot of complex UI, Corbado provides you with prebuilt and configurable **UI components**.

Corbado provides the UI components in two different packages:

- [React components](/frontend-integration/react)
- [VanillaJS components](/frontend-integration/vanilla-js)

{/*

**Note:** You can use the Corbado [VanillaJS](/frontend-integration/vanilla-js) with other frameworks / technologies that are using Javascript.


| Component          | React                        | VanillaJS                         | Description                 |
| ------------------ | ---------------------------- | --------------------------------- | --------------------------- |  
| [Corbado Provider](/frontend-integration/ui-components/corbadoprovider)   | ```<CorbadoProvider/>```     | ```Corbado.load()```              |Necessary for **adding UI components** to an app. This component is a wrapper that allows you do define global configurations.|
| [Signup](/frontend-integration/ui-components/signup)            | ```<SignUp/> ```             | ```Corbado.mountSignUpUI()```     |Component that allows users to create a new account.|
| [Login](/frontend-integration/ui-components/login)             | ```<Login/>```               | ```Corbado.mountLoginUI()```      |Component that allows users to log in to their account.|
| [Corbado Auth](/frontend-integration/ui-components/corbadoauth)      | ```<CorbadoAuth/>```         | ```Corbado.mountAuthUI()```       |Combines Sign Up and Log In component and takes care of authentication-related navigation.|
| [Passkey List](/frontend-integration/ui-components/passkeylist)       | ```<PasskeyList/>```         | ```Corbado.mountPasskeyListUI()```|Can be embedded in user's profile page and lists all passkeys created by the user.|
