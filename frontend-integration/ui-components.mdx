## Overview

Corbado helps you to add a full passkey authentication process to your application. As all of this involves a lot of complex UI, Corbado provides you with prebuilt and configurable **UI components**.

## Available Packages

Corbado provides the UI components in two different packages:

- [React components](/frontend-integration/react): Can be used for React or React-based frameworks (e.g. [Remix](/blog/remix-passkeys), [Next.js](/frontend-integration/next-js)).
- [VanillaJS components](/frontend-integration/vanilla-js): Can be used for all other frameworks (e.g. [Angular](/frontend-integration/angular), [Vue.js](/frontend-integration/vue-js), [Svelte](/frontend-integration/svelte)).

## Available UI Components

**Note:** You can use the Corbado [VanillaJS](/frontend-integration/vanilla-js) components with any other framework / technology that uses JavaScript in the frontend (e.g. also Python, PHP or Java projects).


| Component          | React                        | VanillaJS                         | Description                 |
| ------------------ | ---------------------------- | --------------------------------- | --------------------------- |  
| [CorbadoProvider](/frontend-integration/ui-components/corbadoprovider)   | ```<CorbadoProvider/>```     | ```Corbado.load()```              |**Required** when adding UI components to an application. <br/>Applies global project configuration (e.g., projectID, themes, etc.) to all Corbado components. Before using any of the following components, ensure the CorbadoProvider is properly set up.|
| [Signup](/frontend-integration/ui-components/signup)            | ```<SignUp/> ```             | ```Corbado.mountSignUpUI()```     |Component that allows users to create a new account.|
| [Login](/frontend-integration/ui-components/login)             | ```<Login/>```               | ```Corbado.mountLoginUI()```      |Component that allows users to log in to their account.|
| [CorbadoAuth](/frontend-integration/ui-components/corbadoauth)      | ```<CorbadoAuth/>```         | ```Corbado.mountAuthUI()```       |Combines Signup and Login component and takes care of authentication-related navigation.|
| [PasskeyList](/frontend-integration/ui-components/passkeylist)       | ```<PasskeyList/>```         | ```Corbado.mountPasskeyListUI()```|Can be embedded in user's profile page and lists all passkeys created by the user.|

For all UI components, you can customize translations, themes and error screens. Find detailed information in the [Customization](/frontend-integration/ui-components/customization) page.
