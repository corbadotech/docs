## 2. Add web-js to Nuxt.js code

```html App.vue
<template>
  <div id="corbado-auth"></div>
</template>

<script setup>
  import Corbado from "@corbado/web-js";
  import { onMounted } from "vue";

  onMounted(() => {
    //if user is authenticated you can navigate to a protected route
    if (Corbado.isAuthenticated) {
      return;
    }

    const authElement = document.getElementById("corbado-auth"); //Element where you want to render CorbadoAuth UI

    Corbado.mountAuthUI(authElement, {
      onLoggedIn: () => {
        //post login actions can be performed here.
      },
    });
  });
</script>
```

If you need access to user data on the server, also take a look at our [Corbado Node.js SDK](/backend-integration/node-js-sdk).
