## Introduction

This component handles the user login process. It authenticates the user with the passkey and handles fallback scenarios.

Instructions for configuring the information users can utilize for the login process and managing fallback options can be found in the [User Flow Configuration](/overview/configuration/user-flow-configuration).

## Example
<CodeGroup>

```tsx React
import { Login } from "@corbado/react";

export const LoginPage = () => {
  const navigate = useNavigate();

  const onLoggedIn = () => {
    navigate("/");
  };

  const navigateToSignUp = () => {
    navigate("/sign-up");
  };

  return <Login onLoggedIn={onLoggedIn} navigateToSignUp={navigateToSignUp} />;
};
```

```html VanillaJS
<script>
  const authElement = document.getElementById('login');
  Corbado.mountLoginUI(authElement!, {
    onLoggedIn: () => {
      window.location.href = '/';
    },
    navigateToSignUp: () => {
      window.location.href = '/sign-up';
    }
  });
</script>

<div id="login"></div>
```

</CodeGroup>

## API reference

| Name             | Type              | Description                                                                                                                                                                                                                                                                                               |
| ---------------- | ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| onLoggedIn       | callback function | This callback function is called after successful login. You can use it to navigate the user to a specific page inside your application (e.g., `/home`).                                                                                                                                                    |
| navigateToSignUp | callback function | The `Login` component contains a clickable link that allows users to navigate to the page where they can sign up. The `navigateToSignUp` callback will be called whenever a user clicks that link. You should then navigate the users to the part in your application that shows the `Signup` component. |

<Note> This component requires the `CorbadoProvider` component. Most configuration settings will be applied there so make sure to check out the available settings of the [Corbado Provider](/frontend-integration/ui-components/corbadoprovider). </Note>
