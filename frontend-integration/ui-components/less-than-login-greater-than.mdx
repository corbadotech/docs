## Introduction

![Login Component](/images/frontend-integration/ui-components/login/login-component.webp)

The \<Login/> component allows users to log into your application. Currently, it supports logging in with passkeys and email one-time passcodes (OTP).&#x20;

<CodeGroup>

```tsx Typescript
import { Login } from "@corbado/react";

export const LoginPage = () => {
  const navigate = useNavigate();

  const onLoggedIn = () => {
    navigate("/");
  };

  const navigateToSignUp = () => {
    navigate("/sign-up");
  };

  return <Login onLoggedIn={onLoggedIn} navigateToSignUp={navigateToSignUp} />;
};
```

```html VanillaJS
<script>
  const authElement = document.getElementById('login');
  Corbado.mountLoginUI(authElement!, {
    onLoggedIn: () => {
      window.location.href = '/';
    },
    navigateToSignUp: () => {
      window.location.href = '/sign-up';
    }
  });
</script>

<div id="login"></div>
```

</CodeGroup>

### API reference

| Name             | Type              | Description                                                                                                                                                                                                                                                                                               |
| ---------------- | ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| onLoggedIn       | callback function | This callback function is called after successful login. You can use it to navigate the user to a specific page inside your application (e.g., /home).                                                                                                                                                    |
| navigateToSignUp | callback function | The `<Login/>` component contains a clickable link that allows users to navigate to the page where they can sign up. The `navigateToSignUp` callback will be called whenever a user clicks that link. You should then navigate them to the part in your application that shows the `<SignUp/>` component. |

**Note:** This component must be wrapped inside a \<CorbadoProvider/> component. Most configuration settings will be applied there so make sure to check out the available settings [here](./less-than-corbadoprovider-greater-than).
