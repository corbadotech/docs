## Automatic Passkey Creation After Login

Users logging in via desktop browsers follow a standard login flow enriched by the automatic passkey upgradation.

<Steps>
  <Step title="Login with password-autofill">
    - User logs in with their password, which is then autofilled by Apple Passwords/Google Password Manager. 
    - User is asked to verify their identity.
    - Passkey creation process runs in the backend, given that they have successfully verified their identity.
    - Passkey is successfully created, user is notified via a push alert and they redirect to app dashboard.
  </Step>

</Steps>
