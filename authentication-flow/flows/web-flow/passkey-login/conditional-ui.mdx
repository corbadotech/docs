<Steps>
    <Step title="Login via auto-suggested passkey in keyboard">
        - User enters their identifier (email/username) on the login page.
        - Browser displays an autosuggest popup if a matching passkey is available for this site.
        - The user selects a stored passkey, authenticates with biometric/PIN, and is logged in instantly.
    </Step>
