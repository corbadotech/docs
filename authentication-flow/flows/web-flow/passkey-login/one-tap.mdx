<Steps>
    <Step title="Users Login via One-Tap">
        - Browser detects user had a passkey linked to the platform. User taps “Login with Passkey” button on the login screen to start authentication.
    </Step>
    <Step title="Automatic OS-native biometric modal appears">
        - System prompts for available saved passkey using browser or OS-native modal (Touch ID).
        - User verify their identity using their screen lock (e.g., fingerprint, PIN, or face recognition).
    </Step>
    <Step title="Access granted">
        - User is seamlessly logged in and redirected to the dashboard.
    </Step>

</Steps>
