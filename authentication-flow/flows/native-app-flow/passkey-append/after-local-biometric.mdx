## Passkey Creation After Local Biometric Login

Users logging into a native app using their local biometric authentication are guided through the passkey creation flow.

<Steps>
     <Step title="Login app using local biometric, followed by passkey append">
        - User first authenticates locally using biometrics set up on their device (e.g., Touch ID, Face ID or PIN).
    </Step>
    <Step title="Passkey append process initiates">
        - Once the system verifies user identity, the user is offered an option to create a passkey for future logins.
    </Step>
    <Step title="OS-native biometric modal appears">
        - User chooses and continues to set up a passkey using fingerprint, face, or PIN for secure sign-in.
    </Step>
    <Step title="Passkey creation successful">
        - User successfully creates passkey and is now linked with the user account for the respective native app.
    </Step>

</Steps>
