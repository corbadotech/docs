     <Step title="Login app using local biometric, followed by passkey append">
        - The user first authenticates locally using biometrics set up on their device (e.g., Touch ID, Face ID or PIN).
        - After this successful local biometric check, the system prompts the user to create or append a passkey.
        - User follows the process and successfully creates a passkey.
