## Passkey Creation on Second Device

Users logging in native app from a new device, follows a standard hybrid login flow and experience passkey creation on the current device.

<Steps>
     <Step title="Login on current device using another device">
        - User enters their identifier (email/username) on the login page.
        - OS detects passkey linked to this account on another device.
        - User scans the QR code/ bluetooth connect and authenticates on the device having passkeys.
        - After successful authentication, user is asked to add a passkey on the current device as well.
        - User continues to create passkey. After successful passkey creation on the current device, the user is asked to setup a PIN and for biometrics enabling.
    </Step>

</Steps>
