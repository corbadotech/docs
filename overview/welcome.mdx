| ------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------- |

## User experience ‚ú®

## Security üîí

In addition to an improved user experience, passkeys better protect your users with [strong two-factor authentication in a single step](https://www.corbado.com/blog/passkeys-vs-2fa-security). When a user touches their fingerprint reader or completes facial recognition, they are simultaneously asserting _something they are_ and _something they have_. Biometric data never leaves the user's device, and Corbado relies on a combination of hardware-level security and public key cryptography to keep users safe. Read more about security [here](/other/security-and-privacy).
