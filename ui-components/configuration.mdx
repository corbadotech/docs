## Customization Options for UI Components

- [CorbadoProvider](/ui-components/configuration/corbadoprovider): This component needs to be initialized before
any other UI components on your page. You will use it for all configuration options.
- [Custom translation](/ui-components/configuration/translation): Customize the text of the UI components to match your
application's language or wording.
- [Custom styling](/ui-components/configuration/styling): Customize the appearance of the UI components to match your
application's design.
