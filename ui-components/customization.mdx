## Customization Options for UI Components

Corbado provides the following customization options:

-   [1. Custom translation](/ui-components/customization/translation)
-   [2. Custom styling](/ui-components/customization/styling)

Customization options are configured in the [CorbadoProvider](/ui-components/corbadoprovider).
